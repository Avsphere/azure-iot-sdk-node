Class: Amqp

azure-iot-amqp-base. Amqp

Basic AMQP functionality used by higher-level IoT Hub libraries. Usually you'll want to avoid using this class and instead rely on higher-level implementations of the AMQP transport (see azure-iot-device-amqp.Amqp for example).
